# 属性编辑器 #

属性编辑器是数据类型指向的编辑器。用户在 Umbraco 后台定义的一个数据类型指向的是一个属性编辑器。在 Umbraco 中，一个属性编辑器由一个 JSON 清单文件和相关的 JS 文件所定义。
当你创建了一个数据类型，你是为数据类型从"属性编辑器"列表中选择了一个属性编辑器（如下图所示）
